# aws-textract

Neste projeto uma imagem é inserida pelo usuário na linha de comando, o script lê e processa a imagem e envia para o aws textract que lê o texto da imagem e devolve para o script imprimir o texto.

Desafio interessante no sentido de configurar a conexão com a aws, realizar o envio de uma imagem lida em Python e fazer a chamada para receber o texto lido pela aws textract.
